
スプライト2倍拡大 4倍拡大 テスト

プログラミングにより画像の伸張を行っています。
もし、ゲーム中に使用するとしたら、重たい処理ですのでゲームループ中には使えません。
ステージ開始前の画面暗転中に処理します。

*********************************************************************************
仕組み

関数:スプライト2倍拡大処理
set_2x2(srcadr,vadr)
srcadr   画像データのアドレス
vadr     vram上の転送先アドレス($0000-$ffff)


"srcadr"は、 incbinやincsprで読み込んだ名前は直接使えません。
farアドレスなためです。
near領域つまりramにいったん転送してから
そのアドレスを指定してください
スプライト1枚の転送量は64ワード($40word)です.

#incspr(spr_01,"sprite.pcx",0, 0, 2,4)

	for( i=0 ; i<64 ; i++ )
		buf16x16[i] = spr_01[i];

	set_2x2(buf16x16,0x5100);    /* 2x2Sprite   set to VideoRam */

*********************************************************************************
x4倍
set_2x2(srcadr,vadr)の転送先をvramではなくramにします。
そしてそれをx2することにより4倍になります。

x4倍したスプライトは、
2x4拡張スプライト2枚ということになります。

*********************************************************************************
単位

スプライト一枚は$40ワード、
ワード単位はvramのアドレス計算でも同じ値です。

	Sprite 1x1 $40 word
	       2x2 $100 word
	       2x4 $200 word
